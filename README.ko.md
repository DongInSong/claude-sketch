## 👀 보이는 것

- 👣 **발자국** — 저장소의 모든 파일. 칠해짐 = 밟음, 점선 = 스쳐감, `⋯16⋯` = 근처도 안 감. 커버리지는 그 세션이 실제로 작업한 저장소 기준 — 추적 파일에 무시되지 않은 파일까지.
- 🎯 **주목 점유율** — **면적 = 도구 호출 수**인 트리맵. 파일 하나가 화면 1/3을 먹고 있으면 그림 자체가 경고.
- 🔥 **파일 히트** — 5분 / 1시간 / 세션 전체. 읽고-읽고-고치고-돌리고… 아니면 읽고-읽고-읽고-읽고. 😬
- 📂 **폴더 전환** — Claude Code가 작업한 적 있는 모든 폴더를 헤더에서 고릅니다(경로는 기록에서 복원). 재시작 필요 없음.
- 🧠 **에이전트** — `main ─task▶ 서브에이전트`. 모델·지시문·상태·소요시간까지. 클릭하면 그 에이전트만 필터.
- 📝 **여백 메모** — "6회 재읽기, 이미 컨텍스트에 있을 듯", "`vendor/` 9회 찌름", "수정 → 실행 루프 👍".
- ✂️ **삭제 & 증감** — Bash의 `rm`은 취소선으로, 편집은 `+12 −3`으로.
- ✳️ **밝게 / 어둡게** — 제목 옆 ✳ 를 누르세요. 누르기 전까진 시스템을 따르고, 한 번 더 누르면 도로 시스템에 맡깁니다.

## 💸 다들 놀라는 숫자

```
새 입력 485 · 캐시 8.4M (100% 재사용) · 출력 10.6k · API 요금 환산 약 $0.03
```

Claude는 매 턴 컨텍스트 전체를 캐시에서 재생합니다. "입력 840만 토큰" 😱 은 의미 없고,
실제로 돈이 되는 건 **485**입니다. 단가는 `~/.claude-sketch.pricing.json`에서 수정.
구독 요금제는 토큰당 과금이 아니니 청구액이 아니라 *작업의 크기*로 읽으세요.
실패한 호출도 셉니다 — 순수 낭비니까요. ❌

## 🎛️ 옵션

| | |
|---|---|
| `-p, --project <dir>` | 관찰 대상 (기본: 현재 폴더) |
| `--port <n>` | 기본 4517 — **막혀 있으면 다음 빈 포트로 이동** |
| `--strict-port` | 이동 없이 그냥 실패 |
| `-d, --detach` | 백그라운드로 띄우고 터미널 돌려주기 |
| `--host <addr>` | 기본 `127.0.0.1` (다른 값이면 크게 경고 ⚠️) |
| `--open` / `--no-open` | 브라우저 열기 / 안 열기 |
| `-v, --version` · `-h, --help` | 🙂 |

## ⚙️ 원리

Claude Code는 이미 모든 이벤트를 `~/.claude/projects/<슬러그>/<세션>.jsonl`에
(서브에이전트는 `<세션>/subagents/**`에) 씁니다. `CLAUDE_CONFIG_DIR`를 설정했다면 그쪽입니다.
claude-sketch는 그 파일을 감시·tail 해서
`tool_use` + `message.model` + `message.usage`를 뽑고 SSE로 브라우저에 흘립니다.

**추정이 아니라 실측.** DB·빌드·프레임워크 없음, **의존성 0**. 📦

## ⚡ 성능

| | |
|---|---|
| 유휴 CPU (브라우저 연결) | 코어의 **0.06%** |
| 메모리 | 약 60MB (최악 105MB) |
| 260MB 기록 최초 파싱 | **1.7초** |
| 도구 호출이 브라우저에 닿기까지 | **41ms** (병렬 8 에이전트에서 최악 121ms) |
| 폴링 엔드포인트 (`/api/sessions`) | 1~5ms |
| 115세션 프로젝트로 전환 | 50ms, 이후 약 1ms |
| 브라우저 전체 리렌더 | 약 30ms, 힙 4MB |

## 🔒 프라이버시

중요한 곳은 읽기 전용 — 프로젝트나 `~/.claude` 아래에는 아무것도 쓰지 않습니다.
쓰는 파일은 임시 폴더의 마커 하나뿐이고, 재시작 때 탭이 이미 떠 있는지 알기 위한 것입니다.
`127.0.0.1`에만 바인딩하고, 도메인 이름으로 오거나 다른 오리진에서 온 요청은 거부합니다.
파일 클릭 시 **먼저 물어봅니다**. 외부 요청 0건 — 폰트까지 전부 번들이라 비행기에서도 돕니다. 🏠
